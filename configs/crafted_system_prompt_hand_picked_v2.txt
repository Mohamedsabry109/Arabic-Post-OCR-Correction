CRITICAL: Your output must be ONLY the corrected Arabic text. No preamble, no explanation, no notes, no markdown. Raw corrected text only.

<task>

If it contains a clear visual error (wrong dots, wrong letter shape, merged/split letters), emit the corrected form. Never skip words, never reorder words, never insert new content.
</task>

<rules priority="absolute">
1. COPY-FIRST PRINCIPLE: Treat every input word as correct unless you see a clear visual OCR error. If a word is a valid Arabic word -- even if uncommon, archaic, or grammatically unexpected -- do NOT change it. You are correcting OCR visual errors, not editing Arabic prose.
2. STRICT VISUAL CORRECTION ONLY: Correct only visual OCR misreads (wrong dots, wrong letter shape, merged/split letters). NEVER substitute words with synonyms, swap prepositions, or rephrase. If a word is grammatically incorrect but visually readable, leave it exactly as is.
3. PRESERVE WORD COUNT AND ORDER: Your output must have the same number of words in the same order as the input. Do not add words, remove words, merge words, or split words unless correcting an obvious OCR merge/split error.
4. DO NOT REWRITE FORMULAIC PHRASES: Religious phrases like "صلى الله عليه وسلم", "رضي الله تعالى عنها", "تعالى", and narrator chains (isnad) must be copied EXACTLY from the input. 5. PRESERVE PREFIXES AND SUFFIXES: Never strip or add conjunctive prefixes (فـ وـ بـ لـ كـ) or suffixes. "فقال" and "قال" are different words -- copy whichever the input has. "شيئا" and "شيء" are different -- copy whichever the input has.

7. HANDLE REPETITION CAREFULLY: Only collapse a repeated phrase if it repeats MORE THAN 3 consecutive times (indicating OCR hallucination). If a phrase appears 2-3 times, it may be intentional -- keep it. When collapsing, output the phrase exactly ONCE.
8. DELETE LATIN ARTIFACTS: Delete any Latin/English characters or words -- these are OCR artifacts, not part of the source text.
9. PRESERVE CLASSICAL SPELLING: Maintain classical or archaic spellings common in historical or religious texts. Do not modernize spelling. Preserve Uthmani script forms.
10. DO NOT add or remove the definite article "ال". If the input has "ال", keep it. If it does not, do not add it.
11. UNRECOGNIZABLE TEXT: If a word is damaged beyond recognition and no correction can be deduced with certainty, leave it unchanged.

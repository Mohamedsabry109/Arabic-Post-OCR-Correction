Arabic optical character recognition (OCR) engines produce systematic errors
rooted in the visual properties of the Arabic script: dot-group character
confusions, morphologically valid but contextually wrong substitutions, and
occasional catastrophic repetition artefacts. Open-source engines such as
Qaari fall short of closed-source alternatives, motivating the question of
whether post-OCR correction via a large language model (LLM) can close this
gap without modifying the recognition engine.

This thesis presents a systematic evaluation of prompt-engineering strategies
for Arabic post-OCR correction using Qwen3-4B-Instruct-2507, an open-source
4-billion-parameter instruction-tuned model. Seven knowledge augmentation
conditions are evaluated across nine datasets --- eight typewritten fonts from
the PATS-A01 corpus and one handwritten corpus (KHATT) --- each changing
exactly one variable relative to a zero-shot baseline: (1)~character confusion
matrix injection (Phase~3); (2)~Arabic orthographic rule augmentation
(Phase~4A); (3)~few-shot examples from the QALB corpus (Phase~4B);
(4)~CAMeL Tools morphological post-processing (Phase~4C); and
(5)~self-reflective prompting from training-split error statistics (Phase~4D).
A combination and ablation study (Phase~5) tests pre-selected multi-component
systems. Finally, Phase~7 applies DSPy-based automated prompt optimisation to
discover whether machine-optimised prompts can outperform hand-crafted
strategies.

All eight PATS-A01 fonts share 2,766 text lines each, drawn from the same
underlying content. An aligned 85/15 train/validation split ensures that every
font receives the same 415 validation samples, so that font is the only
variable when comparing OCR and correction quality across typefaces. KHATT
provides a complementary handwritten evaluation with substantially higher
baseline error rates.

Results reveal that correction effectiveness is strongly font-dependent.
On typewritten fonts with moderate-to-high baseline CER (\tbd--\tbd\%),
zero-shot LLM correction achieves substantial CER reductions of up to
\tbd\% relative improvement. On fonts where Qaari already performs well
(CER below \tbd\%), the LLM tends to introduce more errors than it fixes ---
an over-correction problem driven by the model's tendency to modify tokens
that are already correct. On handwritten text (KHATT), where baseline CER
exceeds \tbd\%, the LLM provides marginal improvement. Knowledge augmentation
shows mixed results across fonts, with no single strategy dominating all
conditions. Phase~5 combinations and Phase~7 DSPy optimisation explore whether
synergistic strategy selection or automated prompt tuning can improve upon
individual approaches.

The central finding is that the optimal correction strategy depends on the
baseline error profile of the input. A one-size-fits-all approach to prompt
augmentation is unlikely to succeed across diverse fonts and script types.
The thesis contributes an experimental framework, a three-stage
export--infer--analyse pipeline, and a comprehensive multi-dataset evaluation
that maps the conditions under which LLM-based post-OCR correction helps,
hurts, or has no effect.
